# Builder Fund

To help expand the Flaunch protocol, we want to help facilitate builders by not only supporting  in our social communities, but by also financially incentivizing development.

* **Timeline:** The fund will run until the budget has been consumed
* **Application periods:** Applications will be reviewed on a per-submission basis with no fixed application windows
* **Grant delivery:** Rewards are usually delivered when code is deployed to mainnet
* **Budget:** 15,000 USDC

### Eligibility

Applications must meet these criteria:

* **TokenManager**: A single contract [TokenManager](manager-types/) that can be added to our TokenManagerFactory. This needs to something that could benefit creators, whether this benefits our existing user base or supports in introducing a new cohort.
* **Flaunch Integrations**: An external protocol that utilises Flaunch to power it's underlying logic.

### Rewards

Rewards will be distributed based on individual review, focussing on high-quality onchain value (e.g. volume) and support of existing flaunched tokens.
