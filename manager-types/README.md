### Creating my Own Manager

Have an idea for your own Token Manager? That's fine! You can make your own, bespoke logic using our existing managers as inspiration and deposit tokens into it directly.

If you want to share your Token Manager with the community, [get in touch](https://x.com/buildonflaunch)! We can help fund an audit and approve the manager on our Token Manager Factory for all to use!
